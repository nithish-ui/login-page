# Rapifuzz

Rapifuzz is a React.js project that includes: 
- A Login Page with email & password input fields. 
- A User Dashboard that fetches users from ReqRes API (https://reqres.in/api/users). 
- Pagination & Search for users.

## Setup
```
npm install
```

## Run
```
npm run dev
```

## Build
```
npm run build
```
